# Welcome to Refined Blog

Welcome to the new **Refined** website! This blog uses a simple Markdown system without any heavy frameworks.

## Why this setup?

- **Privacy First:** Zero trackers, no external CDNs, self-hosted libraries.
- **Simplicity:** Just drop a `.md` file in the `blog/` folder and add it to `index.json`.
- **Speed:** Pure HTML, CSS, and JS.
